About

Hi, my name is Jeremy.

I am an educator-turned-software-developer.

I am fascinated with the ways that technology can be harnessed to empower people.

Read More...

My love for coding began as an undergraduate building small web sites with HTML and CSS to enhance class projects which eventually led to teaching myself Javascript as a way to puzzle out interesting programming problems. As a teacher, my goal was to help students turn from consumers of technology to using technology to realize their creative potential. The students I worked with shared stories with other children around the world, created blogs and videos, and eventually built their own electronic toys using Arduino and the Scratch programming platform.

Currently, I am making a career transition from teaching to software development. My past experience in designing curricula and educational programs provides me with extensive practice in creative problem-solving which will help me in a career in software development. These projects involved logistics problems, planning with specific outcomes in mind, as well as delegating tasks and sharing accountability with a team. I hope to bring this experience with organization, design, and learning to enhance a software development team.

Scoring

Scoring is a web-based tool for teachers to organize and visualize formative assessment data.

Users can create and store student lists organized by class roster, organize assessments given over a marking period, generate lists of students sorted by level of performance, and visualize class and student performance data.

Scoring uses AngularJS for views, models, and API calls. User data is stored using Firebase. Charts are generated from data using the D3 library.

Reading Attitudes Survey

The Elementary Reading Attitudes Survey is an electronic implementation of the Elementary Reading Attitudes Survey by Michael C. McKenna and Dennis J. Kear.

With this app, teachers can measure students' feelings toward academic and recreational reading which are predictors of success in reading in elementary school. Teachers create class rosters which generate a random key code for students to access the survey. Students can take the survey and the results are calculated and stored in a database and available to the teacher to evaluate.

The Reading Attitudes Survey is created with AngularJS, Firebase, and Bootstrap with Sass.

Local Weather Viewer

The Local Weather Viewer lets a user view current, 3-day, and 5-day forecasts for weather in any US zip code.

The Local Weather Viewer uses the Open Weather Map API to retrieve weather data. The user interface is created with Bootstrap 4, Sass and using SVG icons from Meteocons triggered by API data. Jquery code is encapsulated using Browserify.
